# 📱 Pokédex

Una pequeña aplicación de Pokédex construida con **React Native** y la **PokeAPI**, con toda la información traducida: tipos, habilidades, descripciones y más.

---

## 🚀 Funcionalidades

- ✅ Lista de Pokémon (limitada a 50 por defecto)
- 🔍 Búsqueda por nombre
- 🌐 Filtro por tipo
- 📋 Detalles del Pokémon seleccionado:
  - Imagen
  - Tipos (en español)
  - Altura y peso
  - Habilidad principal (en español)
  - Descripción (en español)

---

## 🧠 Tecnologías Usadas

- [React Native](https://reactnative.dev/)
- [PokeAPI](https://pokeapi.co/)
- Estilos con `StyleSheet`
- Fetch API para llamadas HTTP
